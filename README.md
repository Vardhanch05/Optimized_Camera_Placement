# 🎥 Camera Placement Optimizer

An interactive web-based tool to design an area and optimize security camera placement for maximum coverage.

## 🚀 Features
- Draw custom polygon areas
- Place and rotate cameras interactively
- Visualize camera coverage
- Auto-optimize camera placement (greedy algorithm)
- Undo / redo support
- Optional FastAPI backend for optimization

## 🖥️ Frontend
- HTML5 Canvas
- Vanilla JavaScript (ES modules)
- CSS (modern UI)

## ⚙️ Backend (Optional)
- FastAPI
- Python
- Shapely, NumPy

## ▶️ How to Run

### Frontend
Just open `index.html` in a browser.

### Backend

pip install -r requirements.txt
python main.py
